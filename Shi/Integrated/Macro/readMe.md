This txt file is intended to explain each macro's purpose

* Save as TIFF and PNG

when you press s, you will save the image in the current window to its current directory as tiff and png

* Save as PNG

when you press p, you will save the image in the current window to its current directory as png ONLY

* Smooth and calibration

Apply smooth function to the current image window, then add a calibration bar based on its contrast value

* redox\_splitter

Split the redox image channel, the first one will be FAD, the second one will be NADH

* Open and Subtract 845

Open all the oir images under selected directory, subtract 841,843,845 with 862, apply LUT

* Open and Subtract 844.6

Open all the oir images under selected directory, subtract 841.8, 844.6 with 862, apply LUT

* LUT CH Tiff Only

Open all tiff in the selected directory, apply LUT

* Reverse stack and plot Z

when you press z, the selected ROI will be revered and z-plot of the corresponding region will be shown.
